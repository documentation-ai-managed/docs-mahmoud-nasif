## Overview

Stay informed about the latest releases of Mahmoud Nasif, your default documentation space for organizing project docs. This changelog highlights new features, improvements, bug fixes, and any breaking changes. You receive notifications for major updates via the dashboard.

<Callout kind="tip">
  Subscribe to updates in your account settings at `https://dashboard.example.com/settings/notifications` to get email alerts for new releases.
</Callout>

## Recent Releases

<Update label="2024-11-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added support for custom brand colors, like `#3B82F6`, in your documentation themes.
- Introduced `<Video>` and `<Iframe>` components for embedding media directly in MDX pages.

## Improvements

- Enhanced MDX parsing with better special character escaping for `{` and `<` in prose.
- Optimized rendering performance for pages with multiple `<Columns>` and `<Tabs>` components.

## Bug Fixes

- Fixed parsing errors in nested code blocks using 4 backticks.
- Resolved heading hierarchy issues when starting with H2 only.

</Update>

<Update label="2024-10-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- New `<ParamField>` and `<ResponseField>` components for API documentation.
- Added `tags` support to `<Update>` components for better categorization.

## Bug Fixes

- Corrected JSX attribute syntax requiring double quotes for strings.
- Fixed Mermaid diagram rendering by enforcing code block syntax only.

</Update>

<Update label="2024-09-10" description="v1.0.0" tags={["breaking", "feature"]}>

## Breaking Changes

- Removed H1 headings from body content; always start with H2.
- Updated component attributes to require PascalCase and exact syntax.

## New Features

- Initial release with core components: `<Callout>`, `<Card>`, `<Steps>`, `<Tabs>`.
- Full MDX support with YAML frontmatter validation.

<Expandable title="How to Migrate from v1.0.0" default-open="true">

Follow these steps for breaking changes:

<Steps>

<Step title="Update Frontmatter">

Ensure no blank lines inside YAML.

</Step>

<Step title="Fix Headings">

Replace all `# H1` with `## H2`.

```javascript
// Before (wrong)
# Title

// After (correct)
## Title
```

</Step>

<Step title="Escape Specials">

Wrap `{API_KEY}` and `<100ms` in backticks.

</Step>

</Steps>
