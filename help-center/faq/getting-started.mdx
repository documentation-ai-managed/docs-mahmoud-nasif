## Overview

Getting started with mahmoud nasif is easy:

1. Visit `https://mahmoudnasif.com/sign-up`
2. Create your account with email
3. Verify your email address
4. Login to your dashboard
5. Click "New Project" to create documentation for your project

## Docs Structure

You can organize your docs into projects, sections, and pages. Edit using the built-in editor, or import Markdown files.

For more advanced features, explore the "Guides" section.
