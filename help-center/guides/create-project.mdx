## Overview

Start documenting in minutes on mahmoud nasif.

<Steps>
  <Step title="Login to Dashboard">
    Go to `https://mahmoudnasif.com` and sign in.
  </Step>
  <Step title="Create new project">
    Click the "New Project" button. Enter your project name and description.
  </Step>
  <Step title="Organize sections">
    Add sections to split your docs logically. Example: Setup, API, Guides.
  </Step>
  <Step title="Begin writing">
    Use the editor to add content. Save or publish your first page.
  </Step>
</Steps>
